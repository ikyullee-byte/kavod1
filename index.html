엑설런트 라떼 ONLY ICE

  1. 베이스: 우유 110ml + 바닐라시럽 15g + 연유 5g
  2. 토핑: 엑설런트 아이스크림 1개 올리기
  3. 샷 추가: 아이스크림 위로 에스프레소 40g 붓기
  4. 마무리: 숟가락으로 밑부분 살짝 섞어주기

카보드 크림 라떼 ONLY ICE

  1. 베이스: 우유 110ml + 연유 10g + 버터스카치 섞기
  2. 크림: 생크림 25g + 연유 20g 아주 강하게 휘핑 후, 액상 생크림 25g 붓고 잘 섞기
  3. 샷 추출: 원두 20g 강하게 탬핑 후, 잔 돌려가며 15ml 짧고 진하게 추출하여 올리기

자몽 에이드 & 티

아이스 자몽에이드
  1. 잔에 얼음 100g + 자몽 농축액 40ml
  2. 자몽 슬라이스 1개 잔 옆면에 깔기
  3. 탄산수 1개 + 나머지 얼음 채우기
  4. 맨 위에 자몽 퓨레 60g 올리기
핫 자몽티
자몽 농축액 30ml + 자몽 퓨레 60g + 자몽 슬라이스 1개 + 뜨거운 물 220ml

레몬 에이드 & 티

아이스 레몬에이드
레몬청 40g + 레몬즙 10ml + 시럽 10g + (탄산수/얼음 적당량)
핫 레몬티
레몬청 30g + 레몬즙 10ml + 시럽 10g + (뜨거운 물 200~220ml)
