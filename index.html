# ☕ 카보드(Kaboard) 통합 매뉴얼 및 레시피 --- ## ✅ 1. 일일 오픈/마감 체크리스트 > 바쁜 시간대나 봉사자가 교대할 때 누락되는 일이 없도록 확인하는 용도입니다. **☀️ 오픈 (Open)** - [ ] 커피 머신 켜기 및 예열 확인 - [ ] 제빙기 얼음 양 확인 및 전용 스쿱 비치 - [ ] 시럽, 연유, 소스류 펌프 작동 및 잔량 확인 - [ ] 쇼케이스 및 냉장고 온도 정상 작동 확인 (우유, 생크림 등) - [ ] 레몬청, 자몽청, 퓨레 등 냉장 재료 상태 확인 **🌙 마감 (Close)** - [ ] 커피 머신 그룹헤드 청소 및 백플러싱 (마감 약품 사용) - [ ] 그라인더 주변 원두 가루 청소 - [ ] 사용한 집기류(샷글라스, 스팀피처, 스푼 등) 세척 및 건조 - [ ] 남은 우유, 생크림 등 냉장 보관 및 유통기한 라벨링 확인 - [ ] 쓰레기통 비우기 및 주변 정리 --- ## 💡 2. 빠른 계량 팁 (시럽 펌프 변환) > 저울 사용이 어려울 때를 대비한 기준표입니다. (사용하시는 펌프 규격에 따라 다를 수 있으니 한 번 테스트 후 수정해서 쓰세요!) * **일반 시럽 1펌프** = 약 10g ~ 15g * **바닐라 시럽 30g** = 약 2~3펌프 * **연유 5g / 10g** = 티스푼 1스푼 / 2스푼 * **레몬즙 10ml** = 1펌프 (또는 전용 스퀴저 기준 1번) --- ## 📖 3. 카보드 레시피 ### ☕ 커피 (COFFEE) | 메뉴명 | 🔴 핫 (HOT) | 🔵 아이스 (ICED) | | :--- | :--- | :--- | | **아메리카노** | **에스프레소 40g** + 뜨거운 물 200ml | **에스프레소 40g** + 얼음 150g + 차가운 물 140ml | | **카페라떼** | **에스프레소 40g** + 우유 180ml | **에스프레소 40g** + 얼음 150g + 우유 160ml | | **바닐라라떼** | **에스프레소 40g** + 바닐라시럽 30g + 연유 5g + 우유 180ml | **에스프레소 40g** + 바닐라시럽 30g + 연유 5g + 얼음 150g + 우유 150ml | ### ✨ 시그니처 (SIGNATURE) | 메뉴명 | 제조 순서 및 방법 (ONLY ICE) | | :--- | :--- | | **엑설런트 라떼** | 1. **베이스:** 우유 110ml + 바닐라시럽 15g + 연유 5g
